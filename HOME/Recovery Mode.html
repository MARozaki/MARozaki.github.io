Menu Recovery Mode

  1. Reboot system now => Memulai ulang perangkat
  2. Reboot to boot loader => Memulai ulang ke boot pemuat
  3. Apply update from ADB => Menerapkan pembaruan dari ADB
  4. Apply update from SD card => Menerapkan pembaruan dari kartu memory
  5. Wipe data/factory reset => Menghapus data/reset pabrikan
  6. Wipe cache partition => Menghapus cache partition
  7. Mount/system => Menyusun/sistem
  8. Views Recovery log => Meninjau pemulihan log(catatan)
  9. Power Off => Daya Mati

For Samsung J2 Prime

Recovery Mode: Tekan tombol Power + Home + V+ secara bersamaan.

Mode Odin: Tekan tombol Power + Home + V- secara bersamaan.

Mode Aman: Tekan tombol Power + secara bersamaan.
